fuelog-python
=============

Official Python client for the `Fuelog <https://fuelog.app>`_ fuel tracking API.

Covers the full **REST API** and the **MCP (Model Context Protocol) server** —
with zero third-party dependencies, typed models, and a comprehensive test suite.

.. code-block:: bash

   pip install fuelog
